django-openinghours

Covered use cases and examples for django-openinghours.

Test whether a company is currently open

{% verbatim %} 
{% isCompanyCurrentlyOpen 'company-ltd' as company_open %}
{% if company_open %}
	Come in, we're open!
{% else %}
	Sorry, we're closed.
{% endif %}

In version 0.0.27 and earlier, it was done like this:

{% if 'company-ltd'|isCompanyCurrentlyOpen %}
	Come in, we're open!
{% else %}
	Sorry, we're closed.
{% endif %}
{% endverbatim %} {% isCompanyCurrentlyOpen 'company-ltd' as company_open %} {% if company_open %} Come in, we're open! {% else %} Sorry, we're closed. {% endif %}

List company's opening hours

{% verbatim %} 
{% companyOpeningHoursList 'company-ltd' %}

In version 0.0.27 and earlier, it was done like this:

{{ 'company-ltd'|companyOpeningHoursList }}
{% endverbatim %} {% companyOpeningHoursList 'company-ltd' %}
